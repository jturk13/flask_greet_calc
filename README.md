This was a basic project that practicing the basics of flask. There is a calc folder that uses basic mathematical operations to provide different routes for the url.
There is also a greet folder that consists of simple web application with three routes that respond with different messages:
Each route is associated with a specific function (welcome, welcome_back, welcome_home), and when a user accesses the corresponding URL, the associated
function is called, and the respective message is returned as the response.
